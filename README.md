# Portafolio

Proyecto de portafolio personal desarrollado con HTML, CSS y JavaScript.

## Características
- Diseño moderno y responsivo.
- Sección de proyectos con capturas de pantalla.
- Enlaces a redes sociales y contacto.
- Animaciones suaves y estética premium.

## Tecnologías
- HTML5
- CSS3 (Vanilla)
- JavaScript

## Instalación
1. Clona el repositorio:
   ```bash
   git clone https://github.com/TU_USUARIO/Portafolio.git
   ```
2. Abre `index.html` en tu navegador.
